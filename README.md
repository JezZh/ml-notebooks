# ML Notebooks

Cleaned machine learning and computer vision notebooks prepared for portfolio review.

## Notebooks

- `air-quality-forecasting.ipynb` - deep learning models for UCI Air Quality forecasting. Requires preprocessed `air_quality_train.csv`, `air_quality_val.csv`, and `air_quality_test.csv` files uploaded to `/content/` when running in Colab.
- `energy-consumption-forecasting.ipynb` - household energy time-series forecasting with anomaly-aware analysis.
- `jellybean-color-segmentation.ipynb` - RGB, HSV, LAB, K-Means, and hybrid color segmentation.
- `logistic-regression-model-evaluation.ipynb` - classifier evaluation with confusion matrix, precision-recall, ROC, and ROC-AUC.
- `milk-production-time-series-forecasting.ipynb` - ARIMA-style forecasting and stability analysis.
- `missing-pixel-image-reconstruction.ipynb` - missing-pixel image reconstruction workflow.
- `sms-spam-classification.ipynb` - text classification pipeline for SMS spam detection.
- `titanic-survival-prediction.ipynb` - tabular machine learning pipeline for passenger survival prediction.

## Notes

- Large datasets and generated outputs are not included.
- Some notebooks download public datasets directly; the air-quality notebook currently expects preprocessed CSV splits.
